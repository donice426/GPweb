¿De qué trata esta web?

Esta web organiza la documentación técnica de gpVision™: arquitectura del sistema, operación desde la interfaz, configuración (clases/defectos), componentes de hardware/mecánica y una guía práctica de corrección de fallas.

Usa el menú para navegar por módulos. La idea es que el equipo encuentre rápido lo que necesita para operar, diagnosticar o ajustar el sistema sin mezclar temas.
